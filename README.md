__算法（algorithm）和模型（model）的区别：__
* 一定程度上可以理解为过程与结果的关系
   * 例如通过对不同温度下的一系列化学反应进行实验、数据记录以及整理拟合得到阿伦尼乌斯公式
   * 其中通过实验得到和处理数据的过程可以算作算法
   * 而阿伦尼乌斯公式这一经验公式是该算法以化学反应速率随温度等条件的变化为输入得出的模型
* 也可以理解为概括与具体、泛用与特化的关系：
   * 仍以如上的实验及数据处理方法与阿伦尼乌斯
   * 其中通过实验得到和处理数据的过程可以算作算法，除了测算化学反应速率外也可用于探究各类物理变化等其他领域
   * 而阿伦尼乌斯公式仅可用于测算化学反应速率，但在其适用领域内相比算法更节省时间、算力等资源
