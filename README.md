__算法（algorithm）和模型（model）的区别：__
* 一定程度上可以理解为过程与结果的关系
   * 例如通过对不同温度下的一系列化学反应进行实验、数据记录以及整理拟合得到阿伦尼乌斯公式
   * 其中通过实验得到和处理数据的过程可以算作算法
   * 而阿伦尼乌斯公式这一经验公式是该算法以化学反应速率随温度等条件的变化为输入得出的模型
* 也可以理解为概括与具体、泛用与特化的关系：
   * 仍以如上的实验及数据处理方法与阿伦尼乌斯
   * 其中通过实验得到和处理数据的过程可以算作算法，除了测算化学反应速率外也可用于探究各类物理变化等其他领域
   * 而阿伦尼乌斯公式仅可用于测算化学反应速率，但在其适用领域内相比算法更节省时间、算力等资源

思考：在上世纪90年代的人类基因组计划中，为何单一私企的进展速度能持平甚至超过国家牵头的官方组织
   * 受限于Sanger测序法读长有限且通量较低的缺陷，对于约30亿bp的人类基因组，需通过限制性内切酶依次截短基因组DNA片段扩增后进行测序再拼接起来
   * 而Celera公司率先采用全基因组鸟枪法————将基因组DNA随机打碎为若干小片段进行Sanger测序，再利用计算机根据各片段间的重叠部分将其连接起来。该方法效率更高
   * Celera公司和全基因组鸟枪法在人类基因组计划中取得的成功也凸显出计算机技术在生物学研究中的巨大作用和潜力
