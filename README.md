算法（algorithm）和模型（model）的区别：
两者一定程度上可以理解为过程与结果的关系：
也可以理解为概括与具体、泛用与特化的关系：
例如化学家通过对不同温度下的一系列化学反应进行实验、数据记录以及整理拟合得到阿伦尼乌斯公式，
其中实验得到和处理数据的过程可以算作算法，而阿伦尼乌斯公式这一经验公式是该算法以化学反应速率随温度的变化为输入得到的模型
该模型在其他领域并不适用，而该算法可以通过输入其他领域的相关数据产出其他结果作为模型
